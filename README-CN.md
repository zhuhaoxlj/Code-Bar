> 该项目的 Java Swing 版本已经停止更新，基于 PySide6 的重制版 Code Bar 正在开发，敬请期待

## 介绍

Code Bar 是一款可以快速搜索调用常用代码片段的工具，同时也可以管理你的代码片段。

- [ ] PySide6 重制
- [ ] 代码片段云同步
